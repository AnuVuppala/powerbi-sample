# Power BI Dashboard Mockup — Loan Default Risk Analysis

This dashboard visualizes borrower risk patterns using KPIs, credit score buckets, EMI burden, and a ranked list of high-risk borrowers. 
The goal is to demonstrate BI dashboard design skills relevant to fintech analytics and collections strategy.

## Included Visuals:
- KPI cards: Total Borrowers, Default Rate %, Avg Credit Score, Avg EMI Ratio
- Bar chart: Default Rate by Credit Score Range
- Scatter plot: EMI Ratio vs Default Probability with an annotated insight
- High-risk borrower table with multi-factor scoring
- Loan Duration slicer mockup for interactivity
